Edit Button Test Suite

This page lists all the tests you can use to make sure the Edit Button appears and identifies the correct links.

  • Page has only a application/x-wiki link tag.
  • Page has only a rel=edit in an a tag. The rel=edit destination is on the same origin.
  • Page has only a rel=edit in an a tag. The rel=edit destination is on a different origin.
  • Page has only a rel=edit in a link tag.
